Support

To ask any question about Vibe, please visit Google Groups. And to report a bug, share an idea for a new feature or commit your work, please use the appropriate GitHub Issues.

Professional Support

Vibe is, and forever will be, open source as Vibe is distributed under the Apache 2 license, one of the most flexible open source licenses. But because choosing a technology like Vibe is a kind of long-term investment, you need to be convinced of a better expected value for that investment. That's why, we offer commercial support.

The creator of Vibe offers professional support for Vibe Protocol, Vibe Java Server, Vibe JavaScript Client and Vibe Java Platform through Async-IO, the official sponsor of Vibe. Please visit Async-IO for the details if you are interested. You can also propose your own request like adding custom feature or prioritizing specific features.
