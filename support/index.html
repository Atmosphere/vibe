Support

As support channels, we use the Groups and StackOverflow tagged with the vibe tag. For a complicated issue, please make sure to specify the followings. The less informations you provided, the lower priority it will get.

  • Vibe project version
  • What you were trying to
  • Bare minimum steps or code to reproduce
  • Result with the latest version
  • Result with older version
  • Anything where you think it's relevant
