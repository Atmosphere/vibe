Vibe JavaScript Client is a concise and lightweight client-side JavaScript library. It not only provides Vibe connectivity on every browser-based and Node-based applications but also focuses on making the best use of realtime connectivity in place of Ajax to help write low-latency and event-driven web applications, not just for chat application.
